# Security Policy

## Reporting a vulnerability

Do **not** open a public GitHub issue for security vulnerabilities.

To report a vulnerability:

- Use the **"Report a vulnerability"** button on the Security tab of this
  repository (GitHub private advisory).
- Or email <babak@cocode.dk>.

Acknowledgement within 5 business days; fix targeted within 30 days of
confirmation.

## Supported versions

This is a static marketing site; only the latest deployed `main` branch is
supported.

| Version | Supported |
|---------|-----------|
| latest  | ✅ |
| older   | ❌ |

## Scope

In scope: anything reachable at
`https://cocodedk.github.io/klinik-for-manuel-terapi/`.

Out of scope:

- The Planway booking widget at `klinik-for-manuel-terapi.planway.com` —
  report directly to Planway.
- The clinic's email and phone — those are direct contact channels, not
  application endpoints.
